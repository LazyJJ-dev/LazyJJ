## Quick Install

```bash
curl -fsSL https://lazyjj.dev/install.sh | bash
```

## Why LazyJJ?

Jujutsu (JJ) is powerful but requires configuration. **LazyJJ gives you a complete stacked workflow out of the box.**

## Features

<CardGrid stagger>
  <Card title="Stack Commands" icon="list-format">
    Navigate and manage stacks with `stack-view`, `stack-top`, `stack-sync`.
    Create stacked PRs with `pr-stack-create`.
  </Card>
  <Card title="Claude Integration" icon="rocket">
    Streamlined worktree management for Claude Code. Create isolated workspaces
    with `claude-start`, AI conflict resolution.
  </Card>
  <Card title="GitHub Helpers" icon="github">
    Quick PR access with `pr-view`, `pr-open`. Generate stack summaries. No
    third-party service dependency.
  </Card>
  <Card title="Sensible Defaults" icon="setting">
    Colors, pager settings, UI tweaks. Revset aliases (trunk, stack, stacks).
    Shortcuts for common operations.
  </Card>
</CardGrid>
