
## Features

<CardGrid stagger>
  <Card title="Core Aliases" icon="pencil">
    Shortcuts like `st`, `d`, `n`, `e` plus value-add aliases like `log-short`, `diff-summary`, `git-fetch`.
  </Card>
  <Card title="Stack Workflow" icon="list-format">
    Navigate and manage commit stacks with `stack-view`, `stack-top`, `stack-bottom`, `stack-sync`.
  </Card>
  <Card title="GitHub Integration" icon="github">
    Create and manage stacked PRs with `pr-view`, `pr-open`, `pr-stack-create`.
  </Card>
  <Card title="Claude Integration" icon="rocket">
    AI-assisted development with `claude-start`, `claude-stop`, `claude-review`.
  </Card>
</CardGrid>
